# MXCore
PHP cryptocurrency

PoW sha256

# Requisites

- Apache web server
- PHP 7.0 or higher
- PHP Extensions:
  - php_sqlite3
  - php_bcmath
  - php_curl
  
# Public Explorer MXCoin

https://blockchain.mataxetos.es/

# How run
- Clone repository on root website folder
- Navigate into bin folder

For miner node:
  - ./node_miner.sh

For viewer node:
  - ./node_viewer.sh
